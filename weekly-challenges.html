Designing a Call to Action (CTA) Button for a Website and Mobile Apps

748 Designers enrolled
The aim of this challenge is to design a Call to Action (CTA) button that is both effective and aesthetically fitting for a fictional website. Participants should focus on how the button can attract users' attention and encourage them to take a specific action, such as subscribing to a newsletter, downloading an app, or making a purchase.

  • Business Requirements:

    • Usage Context: The CTA button should be designed for an e-commerce website in the fashion industry.
    • Action Incentive: The button should clearly communicate to users that clicking it will lead to a purchase.
    • Design Integration: The button must be aesthetically aligned with the overall style of the site, maintaining color and typography consistency.
    • Accessibility: The button must be easily visible and accessible to users, including those with visual impairments.

  • Technical Specifications:

    • Size: The button should be large enough for easy clicking but not dominate the page.
    • Color: The color choice should provide good contrast with the website’s background and draw user attention.
    • Button Text: A short, concise action phrase, e.g., “Buy Now”.
    • Font: The font should be legible and well-matched with the site’s overall style.
    • Placement on the Site: The button should be placed intuitively and within easy reach of the user.
    • Interactivity: The button should include subtle hover effects (e.g., color change, shadow).

  • Expected Outcome:

    • Button Visualization: A graphic representation of the CTA button in the context of a website.
    • Design Rationale: A brief description of design decisions, including color and font choices, and button placement.
    • Accessibility Analysis: A description of how the button design considers accessibility for various users.
    • Optional: An interactive prototype of the button, demonstrating interactive effects.

  • Evaluation Criteria:

    • Compliance with business and technical requirements.
    • Creativity and innovation in design.
    • Aesthetic appeal and consistency with the assumed website style.
    • Effectiveness in communicating the action to the user.
    • Accessibility and usability of the button.
